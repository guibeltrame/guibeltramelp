# Tailwind UI Craftsman

## Objetivo

Construir interfaces visuais com Tailwind CSS de forma consistente, escalável, legível e performática, respeitando a arquitetura de uma landing page moderna.

## Contexto do projeto

- Stack: Nuxt 3, Vue 3, Tailwind CSS
- Idioma visual do site: limpo, claro e objetivo
- Foco em landing page responsiva com alta qualidade de UX

## Instruções

Ao estilizar qualquer tela ou componente:

1. Adote abordagem mobile first
2. Use espaçamento consistente
3. Mantenha tipografia legível e bem hierarquizada
4. Evite excesso de efeitos visuais
5. Priorize clareza, contraste e escaneabilidade
6. Faça layouts simples e previsíveis

## Convenções de uso

- Use Tailwind como padrão principal de estilização
- Evite CSS customizado quando Tailwind resolver bem
- Evite classes excessivamente longas sem necessidade real
- Preserve consistência entre seções semelhantes
- Mantenha responsividade clara entre breakpoints

## Boas práticas

1. Layout:
   - Containers bem definidos
   - Espaçamento vertical coerente
   - Grid e flex apenas quando fizer sentido
   - Evitar alinhamentos improvisados

2. Tipografia:
   - Headings com destaque claro
   - Textos com boa leitura
   - Tamanhos adequados para mobile e desktop
   - Boa relação entre títulos, subtítulos e corpo

3. Componentes:
   - Botões consistentes
   - Inputs consistentes
   - Cards previsíveis
   - Estados de hover, focus e disabled bem definidos

4. Responsividade:
   - Testar mentalmente mobile primeiro
   - Ajustar para tablet e desktop sem quebrar proporções
   - Evitar blocos que dependam apenas de desktop para funcionar bem

## Regras obrigatórias

- Não usar Tailwind apenas por estética; usar com intenção
- Não prejudicar semântica por causa do visual
- Não criar interface pesada, poluída ou confusa
- Não introduzir padrões visuais diferentes sem necessidade

## Checklist de validação

Antes de finalizar:

- O layout está limpo e consistente
- A interface está responsiva
- Os componentes seguem padrão visual coeso
- O foco visual da página está claro
- A implementação em Tailwind está objetiva
- A UI não prejudica SEO, acessibilidade ou performance
